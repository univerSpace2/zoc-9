# ZOC9 메뉴 맵 (v1)

- 문서 목적: 화면 구조/내비게이션/권한별 액션 범위를 단일 소스로 정리.

## 1. 로그인 전
### 1.1 시작(랜딩)
- 로그인 진입
- 회원가입 진입
- 초대 링크 유입 시 초대 확인 화면으로 이동

### 1.2 회원가입
- 입력: 이메일(필수), 이름(필수), 전화번호(필수), 계좌번호(선택)
- 완료 후 그룹 선택 화면 이동

### 1.3 로그인
- 이메일/비밀번호 로그인
- 성공 시 그룹 선택 화면 이동

### 1.4 초대 확인/수락 (로그인 전 진입 가능)
- 표시: 그룹 이름, 초대한 사람, 부여 권한, 만료 여부
- 로그인/회원가입 완료 후 수락/거절 가능

## 2. 로그인 후 공통
### 2.1 그룹 선택
- 내 그룹 목록
- 그룹이 1개면 자동 입장
- 초대 코드 입력(선택)
- 받은 초대 목록(대기/만료/거절/수락)

### 2.2 내 정보(프로필)
- 이름/전화번호/계좌번호 수정
- 비밀번호 변경

## 3. 그룹 입장 후 (그룹 홈) - 하단 탭 4개
### 탭 A. 모임
- 모임 목록(예정/진행중/완료)
- 모임 생성
- 모임 상태 변경
- 모임 입장

### 탭 B. 매치
- 진행중 모임이 있으면 해당 모임 매치 목록 바로가기
- 오늘 진행중 매치의 빠른 진입 허브

### 탭 C. 멤버
- 멤버 목록
- 권한 기반 멤버 초대/추가/수정/삭제
- 전화번호/계좌번호 클립보드 복사
- 포지션별 승률 표시(기본 요약)

### 탭 D. 더보기
- 그룹 설정: 이름 변경, 공지 관리, 그룹장 위임
- 권한 관리: 역할별 기본 범위 + 핵심 권한 토글
- 초대 관리: 생성/목록/취소/재발급
- 구장 관리: 목록/등록/수정/삭제, 예약 URL(선택)

## 4. 모임 입장 후 (모임 홈) - 하단 탭 3개
### 탭 1. 매치
- 매치 목록
- 매치 생성
  - 인원 구성(2vs2/3vs3/4vs4)
  - 심판 지정(선택)
  - 경기 방식(단판/3판2선승/5판3선승)
  - 목표 점수(기본 15)
  - 듀스 여부(기본 적용)
  - 벌칙 텍스트
  - 팀/포지션 구성
- 매치 상세(세트 목록)
- 세트 시작
- 세트 플레이(득점 입력/로그/종료)

### 탭 2. 통계
- 모임 기준 승률 랭킹
- 멤버별 승/패 기본 요약

### 탭 3. 모임 정보
- 구장/날짜/시간/참여 멤버
- 모임 상태
- 모임 완료 버튼(권한 기반)

## 5. 완료 기록 열람 전용 모드 (공통)
1. 모임 또는 매치가 완료 상태면 기본적으로 수정 버튼을 숨긴다.
2. 예외적으로 `edit_completed_records` 권한이 있는 사용자만 수정 UI를 볼 수 있다.
3. 예외 수정은 감사로그 기록이 전제 조건이다.
